# Minimal Weather App

A new Flutter project.

## Description
This is a simple weather forecasting app that uses the OpenWeather API to fetch the  weather of the user's location. The app displays the weather based on the user's current location, including the city, temperature, and an animation representing the weather. The app supports both light and dark modes for the user interface.

## Features

- Automatic weather retrieval based on the user's current location.

- Weather animations are shown based on the weather condition.

- Light and Dark Mode support for a dynamic background color.

- Uses the OpenWeather API to fetch weather data.

## Requirements

- Flutter (version 2.x or higher)

- Internet access for fetching weather data

- Uses the Geolocator and Geocoding plugins to fetch the user's location.
